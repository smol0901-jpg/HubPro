## 1. ВВЕДЕНИЕ

Система планирования HubPro — это инструмент для эффективного управления задачами, расходами и доходами с возможностью автоматической отправки отчётов в Telegram-группы.

### Основные возможности:
- 📌 Планирование задач на день, неделю, месяц, полгода, год
- 💰 Учёт расходов и доходов
- 📊 Дашборд с статистикой
- 📱 Автоматическая отправка в группы
- ✅ Контроль выполнения задач
- ⏰ Уведомления о предстоящих задачах

## 2. ТИПЫ ПЛАНИРОВАНИЯ

### 2.1 По периоду:

| Тип | Описание | Пример |
|-----|-----------|--------|
| day | На день | Сегодняшние задачи |
| week | На неделю | Планы на текущую неделю |
| month | На месяц | Месячные цели |
| half_year | На полгода | Квартальные задачи |
| year | На год | Годовые цели |

### 2.2 По категории:

| Тип | Описание |
|-----|-----------|
| task | Задачи/дела |
| expense | Расходы |
| income | Доходы |

### 2.3 По статусу:

| Статус | Описание |
|--------|----------|
| pending | Ожидает выполнения |
| in_progress | В процессе |
| completed | Выполнено |
| cancelled | Отменено |

## 3. ТЕХНИКИ ПЛАНИРОВАНИЯ

### 3.1 GTD (Getting Things Done)

**Принцип:**
1. Сбор — запишите все задачи
2. Обработка — определите следующий шаг
3. Организация — распределите по категориям
4. Обзор — еженедельный пересмотр
5. Выполнение — действуйте

**В HubPro:** Создавайте задачи с типом `task`, устанавливайте сроки.

### 3.2 Матрица Эйзенхауэра

| | Срочно | Не срочно |
|---|--------|----------|
| **Важно** | Сделать немедленно | Запланировать |
| **Не важно** | Делегировать | Удалить |

**В HubPro:** Используйте приоритеты и сроки для распределения задач.

### 3.3 Time Blocking (Блоки времени)

**Принцип:** Выделите конкретные временные блоки для определённых задач.

**В HubPro:** Создавайте задачи с точным временем и датой.

### 3.4 Pomodoro Technique

**Принцип:** 25 минут работы → 5 минут отдыха.

**В HubPro:** Создавайте повторяющиеся задачи с интервалом 25 минут.

### 4.1 Создание плана

```javascript
// Пример API
await electronAPI.addPlan({
  title: "Оплатить аренду",
  description: "Перевести арендную плату за май",
  type: "expense", // task, expense, income
  period: "month", // day, week, month, half_year, year
  planned_date: "2026-05-31",
  planned_amount: 50000,
  group_id: 1,
  notify_before: 1440 // уведомить за 24 часа (в минутах)
});
```

### 4.2 Обновление статуса

```javascript
// Отметить выполненным
await electronAPI.updatePlan({
  id: 1234567890,
  status: "completed",
  actual_amount: 48000 // фактическая сумма
});
```

### 4.3 Получение данных

```javascript
// Все планы
const { plans, dashboard } = await electronAPI.getPlanning({
  period: "month",
  type: "expense"
});
```

## 5. ДАШБОРД

Дашборд показывает:

```javascript

  day: [...],      // Планы на день
```
