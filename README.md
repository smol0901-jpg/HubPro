# 🤖 HubPro — Управление Telegram ботами

Профессиональный десктопный центр управления Telegram ботами с шифрованием, авторизацией и автоотправкой сообщений.

[![Version](https://img.shields.io/badge/version-1.0.0-blue)](https://github.com/smol0901-jpg/HubPro)
[![Platform](https://img.shields.io/badge/platform-Electron-green)](https://www.electronjs.org/)
[![Database](https://img.shields.io/badge/database-SQLite-orange)](https://www.sqlite.org/)

---

## 📋 Возможности

- **Управление ботами** — добавляйте, включайте и отключайте Telegram ботов
- **Работа с группами** — привязывайте ботов к чатам для отправки сообщений
- **Чат интерфейс** — отправляйте и получайте сообщения прямо из приложения
- **Расписание** — настраивайте автоматическую отправку сообщений по времени
- **Шифрование** — токены ботов защищены AES-256-CBC
- **Авторизация** — вход по логину и паролю, система пользователей
- **Системный трей** — приложение работает в фоне
- **Автообновления** — автоматическая проверка и установка обновлений

## 🚀 Быстрый старт (для «младенцев»)

### Шаг 1: Установите Node.js

1. Откройте браузер и перейдите по ссылке: https://nodejs.org/
2. Нажмите на большую кнопку **LTS** (слева) — она скачает установщик
3. Запустите скачанный файл (например, `node-v20.x.x-x64.msi`)
4. Нажимайте **Next** → **Next** → **Next** → **Install** → **Finish**
5. После установки откройте **Командную строку** (нажмите `Win + R`, введите `cmd`, нажмите Enter)
6. Введите `node -v` и нажмите Enter — должно написать версию (например, `v20.x.x`)

### Шаг 2: Скачайте проект

1. Перейдите на страницу: https://github.com/smol0901-jpg/HubPro
2. Нажмите зелёную кнопку **Code** → **Download ZIP**
3. Распакуйте архив в удобное место (например, на Рабочий стол)
4. Переименуйте папку в `HubPro` (без пробелов)

### Шаг 3: Установите зависимости

1. Откройте **Командную строку**
2. Перейдите в папку с проектом:
   ```
   cd %USERPROFILE%\Desktop\HubPro
   ```
   (или куда вы распаковали архив)
3. Введите команду и нажмите Enter:
   ```
   npm install
   ```
4. Подождите 2-5 минут, пока устанавливаются библиотеки

### Шаг 4: Запустите приложение

1. В той же командной строке введите:
   ```
   npm start
   ```
2. Подождите 10-30 секунд — откроется окно приложения

### Шаг 5: Войдите в систему

1. В открывшемся окне введите:
   - **Логин:** `NeuralAP`
   - **Пароль:** `0901Admin`
2. Нажмите кнопку **Войти**

**Поздравляем! Приложение запущено!** 🎉

## 📖 Как пользоваться

### Добавление Telegram бота

1. Нажмите **➕ Добавить бота**
2. Введите имя бота (например, «Мой первый бот»)
3. Введите токен бота (получите у @BotFather в Telegram)
4. Нажмите **Сохранить**

**Как получить токен бота:**
1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте команду `/newbot`
4. Следуйте инструкциям, придумайте имя
5. Скопируйте токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Добавление группы

1. Перейдите в раздел **Группы**
2. Нажмите **➕ Добавить группу**
3. Введите название группы
4. Введите Chat ID группы (можно узнать у @userinfobot в Telegram)
5. Выберите бота из списка
6. Нажмите **Сохранить**

### Отправка сообщений

1. Перейдите в раздел **Чат**
2. Выберите группу из списка слева
3. Введите текст в поле внизу
4. Нажмите **Отправить** или клавишу Enter

### Настройка расписания

1. Перейдите в раздел **Расписание**
2. Нажмите **➕ Создать расписание**
3. Заполните:
   - Название (например, «Утренняя рассылка»)
   - Текст сообщения
   - Группу для отправки
   - Дату и время
   - Тип повтора (один раз, ежедневно, еженедельно)
4. Нажмите **Сохранить**

### Управление пользователями (только админ)

1. Перейдите в раздел **Пользователи**
2. Нажмите **➕ Добавить пользователя**
3. Введите логин, пароль и выберите роль
4. Нажмите **Сохранить**

## 🆘 Проблемы и решения

### Ошибка «npm не является внутренней или внешней командой»
- Переустановите Node.js, убедитесь, что отмечена галочка «Add to PATH»

### Приложение не запускается
- Убедитесь, что установлены зависимости: `npm install`
- Проверьте версию Node.js: `node -v`

### Ошибка при добавлении бота «Неверный токен»
- Проверьте токен в Telegram у @BotFather
- Скопируйте токен точно, без пробелов

### Приложение закрывается при нажатии крестика
- Это нормально! Приложение сворачивается в системный трей
- Чтобы закрыть полностью: правый клик по иконке в трее → «Закрыть полностью»
