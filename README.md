# 🤖 HubPro v1.2.0

Профессиональный десктопный центр управления Telegram ботами с шифрованием, авторизацией и автоотправкой сообщений.

---

## 📋 Возможности

- **Управление ботами** — добавляйте, включайте и отключайте Telegram боты
- **Работа с группами** — привязывайте ботов к чатам для отправки сообщений
- **Темы (Topics)** — отправка сообщений в конкретные темы группы
- **Чат интерфейс** — отправляйте и получайте сообщения прямо из приложения
- **Расписание** — настраивайте автоматическую отправку сообщений по времени
- **Шифрование** — токены ботов защищены AES-256-CBC
- **Авторизация** — вход по логину и паролю, система пользователей
- **Экспорт/Импорт** — выгрузка и загрузка данных
- **Системный трей** — приложение работает в фоне
- **Автообновления** — автоматическая проверка и установка обновлений

## 🚀 Быстрый старт

### Установка

1. Скачайте и установите **Node.js** (версия 18+): https://nodejs.org/
2. Скачайте проект: https://github.com/smol0901-jpg/HubPro
3. Распакуйте архив
4. В командной строке перейдите в папку проекта и выполните:
```bash
npm install
npm start
```

## 🌐 Запуск в веб-браузере

Приложение можно запустить в браузере без Electron:

```bash
# Установите зависимости
npm install

# Запустите веб-сервер
npm run web
```

После этого откройте http://localhost:3000 в браузере.

**Примечание:** Для полной функциональности (системный трей, автообновления) используйте десктопную версию.

## 📊 Экспорт и импорт данных

### Экспорт
1. Перейдите в раздел **Настройки**
2. Нажмите **Экспорт данных**
3. Данные сохранятся в формате JSON

### Импорт
1. Подготовьте данные в формате JSON
2. Перейдите в **Настройки**
3. Нажмите **Импорт данных**
4. Выберите файл и подтвердите

## 💬 Отправка в темы (Topics)

### Что такое Topics?
Topics — это внутригрупповые ветки обсуждения в Telegram.

### Как получить ID темы:
1. В группе перейдите в список тем
2. Нажмите на нужную тему
3. Скопируйте ID из ссылки

### Настройка:
При создании группы укажите **ID тем** через запятую: `1,2,3`

## 📖 Как пользоваться

### Добавление бота
1. Нажмите **➕ Добавить бота**
2. Введите имя и токен бота
3. Нажмите **Сохранить**

### Добавление группы
1. Перейдите в **Группы** → **➕ Добавить группу**
2. Заполните название, Chat ID, выберите бота
3. (Опционально) Укажите ID тем через запятую

### Отправка сообщений
1. Перейдите в **Чат**
2. Выберите группу
3. Введите текст и нажмите **Отправить**
