# HubPro (BotHub)

Desktop-приложение для управления Telegram ботами.

## Возможности

- Управление несколькими Telegram ботами
- Привязка ботов к группам/чатам
- Отправка сообщений через реальный Telegram API
- Приём входящих сообщений (polling)
- Системный трей (сворачивание в трей)
- Автообновления через GitHub Releases

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Сборка

```bash
npm run build
```

## Публикация обновлений

```bash
set GH_TOKEN=ghp_ВАШ_ТОКЕН
npm run publish
```
